# Distributed E-Commerce Data Engineering Pipeline

## Project Overview
End-to-end distributed ETL pipeline using Spark, AWS S3, Amazon Redshift and Apache Airflow.

## Architecture
Raw Data → Amazon S3 → Spark ETL → Processed Data (S3) → Amazon Redshift  
Orchestrated using Apache Airflow DAG.

## Technologies
- PySpark
- AWS S3
- Amazon Redshift
- Apache Airflow
- Distributed Data Processing

## Key Features
- Distributed ETL using Spark DataFrames
- Data lake architecture on S3
- Star schema data warehouse design
- Automated workflow orchestration
- Scalable cloud-based architecture
